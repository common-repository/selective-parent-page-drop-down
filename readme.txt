This plugin allows to selectively display pages in the page attributes parent page drop down
== Description ==

If you have lot of pages in your site then it becomes an issue selecting parent pages from a huge drop down.

There is a way to show only selected pages in the drop down as selectable parent pages.

The plugin makes use of page_attributes_dropdown_pages_args filter and allows to select pages to display in the parent page drop-down element.

http://phpwpinfo.com/2016/01/16/wordpress-limit-pages-displayed-in-the-parent-page-drop-down/

== Installation ==

1. Upload and install Selective parent Page Drop Down in page attributes drop down in the same way you'd install any other plugin.
2. A custom meta box called as "Article Options" appears in each article. The field to show the page in the drop down is set to No by default. If you wish to show the page just set it to Yes for the respective page
3. Open the settings screen to select the default behavior or the field (hide or show the parent page)
